    Remove subprocess32 dependency, which doesn't build properly on Windows
    
    Can't have dependencies that don't work cleanly cross-platform.
    When built with MinGW on Windows 7 Home Premium, the subprocess32
    package has build errors.
    
    Note: this is a partial checkin. Calls to subprocess functions may be
    broken in this commit.
