    Fix progress.py to obey PIPENV_COLORBLIND and not use any colors.
    
    Crayon library is used in progress.py to set up BAR_FILLED_CHAR as
    module global, and since progress is imported in core before
    crayons.disable() call takes effect, the BAR_FILLED_CHARS is green even
    though PIPENV_COLORBLIND is set. To make this work as advertised,
    PIPENV_COLORBLIND is checked in progress.py too.
