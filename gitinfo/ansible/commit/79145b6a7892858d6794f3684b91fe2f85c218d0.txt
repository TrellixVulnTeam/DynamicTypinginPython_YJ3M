+.. _set_theory_filters:

+Set Theory Filters
+--------------------
+These mostly deal with set (lists), no filter for union as this already works::
+
+    {{ list1 + list2 }}
+
+To get the intersection of 2 lists (unique list of all items in both)::
+
+    {{ list1 |intersect(list2)}}
+
+To get a unique set from a list::
+
+    {{ list1 |unique }}
+
