    CLN: Remove some dtype methods from API (#23390)
    
    Removes the following from the public API:
    
    * pandas.api.types.is_sequence
    * pandas.api.types.is_any_int_dtype
    * pandas.api.types.is_floating_dtype
    
    xref gh-16163.
    xref gh-16189.
