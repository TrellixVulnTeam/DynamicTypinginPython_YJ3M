+### Python Developer guide
+
+#### Debugging
+
+Install `ipdb`:
+
+```sh
+$ pip install ipdb
+```
+
+Insert a breakpoint in your Python code:
+
+```py
+import ipdb; ipdb.set_trace()
+```
