    fix(ui) Improve display of release files (#10510)
    
    The additional span causes files to be trucated to 0 width if they would
    overflow. By removing the span out `truncate` class can do its job.
    I've made the file column wider as commits can only have a single
    author, and we don't need all that space for a single icon.
    
    I've also converted this component to an es6 class and removed the
    commented out code as it was simple to do.
