    Use event type group display in workflow emails.
    
    This changes the templates that are used to render events in workflow
    emails to use the event type group format. (See: GH-3666.)
    
    This also updates the debug view to display a randomly generated group,
    rather than static content so that there is a variety of different
    possible event types and line lengths.
