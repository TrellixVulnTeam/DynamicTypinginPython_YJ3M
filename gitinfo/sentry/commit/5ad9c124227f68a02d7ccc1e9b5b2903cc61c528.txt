    ref(ui): Allow form search source results to be marked (#11442)
    
    The search map contains the keys `{title, description}`, as well as a
    nested `field` object that contains `label` hand `help`. These are two
    of the same, as the title and description are setup when creating the
    searchMap from the field object.
    
    This enables the title and description to be marked (the matching
    portions highlighted) in the SearchResult.
