    feat(plugins): Fix Sessionstack plugin not working on Issue Details (#12501)
    
    This fixes the Sessionstack plugin on Issue Details. Currently, the sessionstack context type is "default" which causes the ContextsInterface component to render the Default renderer instead of any potential plugins. If type is default, we will try to use "alias" and fall back to "type"
