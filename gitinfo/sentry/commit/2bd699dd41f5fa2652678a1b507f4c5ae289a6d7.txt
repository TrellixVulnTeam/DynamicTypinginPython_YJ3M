    ref(ui) Don't use div inside Button component
    
    Don't use `div` inside the Button component. Doing so makes Button hard
    to use in other block level contexts like `p` elements. By swapping out
    Box for span we also generate fewer DOM nodes overall.
    
    Similarly Icon inside the Button has also been converted to a span
    instead of a Box trimming more DOM.o
    
    I've removed the `alignLabel` prop from `Button` as it appears to be
    entirely unused, and doesn't do anything visually distinct when it is
    used.
