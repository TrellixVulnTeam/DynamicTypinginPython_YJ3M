    feat(ui) Add extra extra small button size (#14684)
    
    The discover sidebar designs call for a button that has less padding
    than the current xsmall button, but more than micro. This button size
    could be useful in the issue saved search as well where we're currently
    using a borderless zero size button which feels a bit too big.
    
    Refs SEN-952
