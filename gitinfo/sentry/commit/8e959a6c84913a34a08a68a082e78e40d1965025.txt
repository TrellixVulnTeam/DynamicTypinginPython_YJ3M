    ref(paths): Remove `:` from paths so repo can be used on Windows (#13382)
    
    Our enhancement configs and some snapshot tests have `:` in their
    paths causing Windows to choke while checking out the repo as
    this is a forbidden character in file paths. This patch
    
    - Renames the files and directories
    - Adds a small shim to enhancement config reader
    - Adds protection to snapshot path generator to prevent future
      instances
