    fix(transactions): Preserve quotes inside of request body (#14756)
    
    Inside the request tab of a detailed transaction view there is a json body that is shown without quotations. Added code to have an option in the ContextData component for preserving these quotes when json is being displayed. This way it can be easily copied and be valid json.
    
    Fixes SEN-998
