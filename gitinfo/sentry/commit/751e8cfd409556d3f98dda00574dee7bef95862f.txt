    feat(breadcrumbs): Add error boundary (#12280)
    
    Add an error boundary around breadcrumb component. Since we don't do a
    lot of validation on breadcrumbs currently, data sent by the user might
    be malformed. This prevents the entire event detail page crashing.
    
    Refs ISSUE-350
