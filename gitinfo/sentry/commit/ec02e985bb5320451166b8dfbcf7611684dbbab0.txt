    fix(ui) Switch remaining logout links to form submit
    
    I missed a few logout links when I switched logout to POST only. I've
    opted to use a hidden form submit with DOM1 event handlers as it
    requires the least amount of Javascript/CSS being added to the django
    templates. I pruned out another logout link that was in a block of
    commented code that we no longer need.
    
    Refs APP-568
