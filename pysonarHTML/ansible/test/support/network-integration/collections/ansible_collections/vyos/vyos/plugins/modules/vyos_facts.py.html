  19 short_description: Get facts about vyos devices.
  20 description:
  21 - Collects facts from network devices running the vyos operating system. This module
  22   places the facts gathered in the fact tree keyed by the respective resource name.  The
  23   facts module will always collect a base set of facts from the device and can enable
  24   or disable collection of additional facts.
  25 author:
  26 - Nathaniel Case (@qalthos)

  29 extends_documentation_fragment:
  30 - vyos.vyos.vyos
  31 notes:
  32 - Tested against VyOS 1.1.8 (helium).
  33 - This module works with connection C(network_cli). See L(the VyOS OS Platform Options,../network/user_guide/platform_vyos.html).
  34 options:
  35   gather_subset:
  36     description:
  37     - When supplied, this argument will restrict the facts collected to a given subset.  Possible
  38       values for this argument include all, default, config, and neighbors. Can specify
  39       a list of values to include a larger subset. Values can also be used with an
  40       initial C(M(!)) to specify that a specific subset should not be collected.
  41     required: false
  42     default: '!config'
  43   gather_network_resources:
  44     description:
  45     - When supplied, this argument will restrict the facts collected to a given subset.
  46       Possible values for this argument include all and the resources like interfaces.
  47       Can specify a list of values to include a larger subset. Values can also be
  48       used with an initial C(M(!)) to specify that a specific subset should not be
  49       collected. Valid subsets are 'all', 'interfaces', 'l3_interfaces', 'lag_interfaces',
  50       'lldp_global', 'lldp_interfaces', 'static_routes', 'firewall_rules'.
  51     required: false
  52 """
