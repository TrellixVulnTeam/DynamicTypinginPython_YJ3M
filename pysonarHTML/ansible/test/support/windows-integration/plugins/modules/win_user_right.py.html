  37     - You can also add special local accounts like SYSTEM and others.
  38     - Can be set to an empty list with I(action=set) to remove all accounts

  40     type: list
  41     required: yes
  42   action:
  43     description:
  44     - C(add) will add the users/groups to the existing right.
  45     - C(remove) will remove the users/groups from the existing right.
  46     - C(set) will replace the users/groups of the existing right.
  47     type: str
  48     default: set
  49     choices: [ add, remove, set ]
  50 notes:
  51 - If the server is domain joined this module can change a right but if a GPO
