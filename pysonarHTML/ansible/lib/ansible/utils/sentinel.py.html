   9 class Sentinel:
  10     """
  11     Object which can be used to mark whether an entry as being special
  12 
  13     A sentinel value demarcates a value or marks an entry as having a special meaning.  In C, the
  14     Null byte is used as a sentinel for the end of a string.  In Python, None is often used as
  15     a Sentinel in optional parameters to mean that the parameter was not set by the user.
  16 
  17     You should use None as a Sentinel value any Python code where None is not a valid entry.  If
  18     None is a valid entry, though, then you need to create a different value, which is the purpose
  19     of this class.
  20 
  21     Example of using Sentinel as a default parameter value::
  22 
  23         def confirm_big_red_button(tristate=Sentinel):
  24             if tristate is Sentinel:
  25                 print('You must explicitly press the big red button to blow up the base')
  26             elif tristate is True:
  27                 print('Countdown to destruction activated')
  28             elif tristate is False:
  29                 print('Countdown stopped')
  30             elif tristate is None:
  31                 print('Waiting for more input')
  32 
  33     Example of using Sentinel to tell whether a dict which has a default value has been changed::
  34 
  35         values = {'one': Sentinel, 'two': Sentinel}
  36         defaults = {'one': 1, 'two': 2}
  37 
  38         # [.. Other code which does things including setting a new value for 'one' ..]
  39         values['one'] = None
  40         # [..]
  41 
  42         print('You made changes to:')
  43         for key, value in values.items():
  44             if value is Sentinel:
  45                 continue
  46             print('%s: %s' % (key, value)
  47     """
  48
