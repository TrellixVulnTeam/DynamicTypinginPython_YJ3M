  10 DOCUMENTATION = r'''
  11 ---
  12 module: raw
  13 short_description: Executes a low-down and dirty command
  14 version_added: historical
  15 options:
  16   free_form:
  17     description:
  18       - The raw module takes a free form command to run.
  19       - There is no parameter actually named 'free form'; see the examples!
  20     required: true
  21   executable:
  22     description:
  23       - Change the shell used to execute the command. Should be an absolute path to the executable.
  24       - When using privilege escalation (C(become)) a default shell will be assigned if one is not provided
  25         as privilege escalation requires a shell.
  26     version_added: "1.0"
  27 description:
  28      - Executes a low-down and dirty SSH command, not going through the module
  29        subsystem.
  30      - This is useful and should only be done in a few cases. A common
  31        case is installing C(python) on a system without python installed by default.
  32        Another is speaking to any devices such as
  33        routers that do not have any Python installed. In any other case, using
  34        the M(shell) or M(command) module is much more appropriate.
  35      - Arguments given to C(raw) are run directly through the configured remote shell.
  36      - Standard output, error output and return code are returned when
  37        available.
  38      - There is no change handler support for this module.
  39      - This module does not require python on the remote system, much like
  40        the M(script) module.
  41      - This module is also supported for Windows targets.
  42 notes:
  43     - "If using raw from a playbook, you may need to disable fact gathering
  44       using C(gather_facts: no) if you're using C(raw) to bootstrap python
  45       onto the machine."
  46     - If you want to execute a command securely and predictably, it may be
  47       better to use the M(command) or M(shell) modules instead.
  48     - The C(environment) keyword does not work with raw normally, it requires a shell
  49       which means it only works if C(executable) is set or using the module
  50       with privilege escalation (C(become)).
  51     - This module is also supported for Windows targets.
  52 seealso:
  53 - module: command
  54 - module: shell
  55 - module: win_command
  56 - module: win_shell
  57 author:
  58     - Ansible Core Team
  59     - Michael DeHaan
  60 '''

  67   raw: cat < /tmp/*txt
  68   args:
  69     executable: /bin/bash
