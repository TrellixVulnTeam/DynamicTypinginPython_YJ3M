  11 DOCUMENTATION = r'''
  12 ---
  13 author: Ansible Core Team (@ansible)
  14 module: include
  15 short_description: Include a play or task list
  16 description:
  17   - Includes a file with a list of plays or tasks to be executed in the current playbook.
  18   - Files with a list of plays can only be included at the top level. Lists of tasks can only be included where tasks
  19     normally run (in play).
  20   - Before Ansible 2.0, all includes were 'static' and were executed when the play was compiled.
  21   - Static includes are not subject to most directives. For example, loops or conditionals are applied instead to each
  22     inherited task.
  23   - Since Ansible 2.0, task includes are dynamic and behave more like real tasks. This means they can be looped,
  24     skipped and use variables from any source. Ansible tries to auto detect this, but you can use the C(static)
  25     directive (which was added in Ansible 2.1) to bypass autodetection.
  26   - This module is also supported for Windows targets.
  27 version_added: "0.6"
  28 options:
  29   free-form:
  30     description:
  31       - This module allows you to specify the name of the file directly without any other options.
  32 notes:
  33   - This is a core feature of Ansible, rather than a module, and cannot be overridden like a module.
  34   - Include has some unintuitive behaviours depending on if it is running in a static or dynamic in play or in playbook context,
  35     in an effort to clarify behaviours we are moving to a new set modules (M(include_tasks), M(include_role), M(import_playbook), M(import_tasks))
  36     that have well established and clear behaviours.
  37   - B(This module will still be supported for some time but we are looking at deprecating it in the near future.)
  38 seealso:
  39 - module: import_playbook
  40 - module: import_role
  41 - module: import_tasks
  42 - module: include_role
  43 - module: include_tasks
  44 - ref: playbooks_reuse_includes
  45   description: More information related to including and importing playbooks, roles and tasks.
  46 '''
