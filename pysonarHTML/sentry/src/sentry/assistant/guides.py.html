   6 # Guide Schema
   7 # id (text, required): unique id
   8 # required_targets (list): An empty list will cause the guide to be shown regardless
   9 #                          of page/targets presence.
  10 # steps (list): List of steps
  11 
  12 # Step Schema
  13 # title (text, required): Title text. Tone should be active.
  14 # message (text, optional): Message text. Should help illustrate how to do a task, not
  15 #                           just literally what the button does.
  16 # target (text, optional): step is tied to an anchor target. If the anchor doesn't exist,
  17 #                          the step will not be shown. if the anchor exists but is of type
  18 #                         "invisible", it will not be pinged but will be scrolled to.
  19 #                          otherwise the anchor will be pinged and scrolled to. If you'd like
  20 #                          your step to show always or have a step is not tied to a specific
  21 #                          element but you'd still like it to be shown, set this as None.
