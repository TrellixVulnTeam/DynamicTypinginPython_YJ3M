  19         List issues to be resolved in a particular release
  20         ``````````````````````````````````````````````````

  22         Retrieve a list of issues to be resolved in a given release.
  23 
  24         :pparam string organization_slug: the slug of the organization the

  34             list(groups),
  35             request.user,

  39         )
